01 · Kanban columns

Three columns is the whole system.

To do, In progress, Done. No custom workflows to configure before you can add your first task — the structure that works for almost everyone is just there on day one.

  • Friendly empty states instead of blank space

02 · Drag-and-drop

Pick it up. Put it down.

Built on SortableJS: grab any ticket and drop it in a new column, or reorder it within the same one. The card animates into place and the new order is written to the database the moment you let go.

03 · Optimistic UI

Nothing waits on the network.

Add a task, check it off, delete it — the screen updates the instant you act. The request to Supabase happens quietly afterward, and if it ever fails, Boardly rolls the change back and tells you.

04 · Command palette

Ctrl+K, then never touch the mouse.

Press Ctrl+K (or Cmd+K) from anywhere on the dashboard. Type a task title to add it straight to To do, or search for an action like jumping to Settings or logging out.

05 · Categories

A colored rail for every kind of task.

General, Work, Personal, Urgent — each gets its own rail color, so a busy board is still readable at a glance, no need to read every title.

06 · Private by design

Row-level security, not just a login screen.

The database itself enforces that you can only ever read or write your own rows — even if someone bypassed the website entirely, the data stays locked to its owner.
